# Google Maps Restaurants & Cafes Scraper

## What does this scraper do?
This is a highly optimized, niche-specific Google Maps scraper designed to extract **Google Maps Restaurants & Cafes ** worldwide. 
It bypasses Google's complex protections automatically and delivers clean, structured data in a simple table format.

## Why scrape Restaurants?
Scraping Restaurants from Google Maps allows you to build highly targeted B2B lead generation lists, enrich your CRM, and perform local market analysis without manual effort.

## Input Parameters
- **locations**: An array of strings representing the cities, zip codes, or areas you want to search in (e.g. `["New York, USA", "London, UK"]`).
- **maxItems**: The maximum number of listings to extract per location.

## Output Data
The actor outputs clean JSON or CSV data containing:
- Business Name
- Phone Number
- Full Address
- Website URL
- Google Maps URL
- Average Rating
- Total Reviews Count
